# OWL UI Components

`OWL UI` is **React Component Design System**, it works in Cepave's monitoring system.


## Install

```sh
$ npm i --save github:cepave/owl-ui#dev
```



## Usage

Includes the style

```html
<link rel="stylesheet" href="dist/owl-ui.css" />
```

Imports `owl-ui.v2`

```javascript
import {Tab, Select, Tip} from 'owl-ui.v2'
```
